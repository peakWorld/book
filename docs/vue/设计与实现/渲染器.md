
# 渲染器的设计
渲染器是框架性能的核心, 渲染器的实现直接影响框架的性能。

## 渲染器与响应系统的结合 7-1
渲染器不仅能够渲染真实的DOM元素, 还是框架跨平台能力的关键。
注: 设计渲染器的时候一定要考虑好可自定义的能力。

## 渲染器的基本概念 7-2
* 挂载: 渲染器把虚拟DOM节点渲染为真实DOM节点的过程。
* 渲染器不仅可以用来渲染,还可以用来激活已有的DOM元素(同构渲染)

## 自定义渲染器
将渲染器设计为可配置的‘通用’渲染器, 实现渲染到任意目标平台。
抽离特定平台的API.
- 设计一个不依赖于特定平台(浏览器)的通用渲染器。7-3
